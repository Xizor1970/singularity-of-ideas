## About

An immersive, research-backed knowledge base exploring the mechanics of intellectual dominance. Why do some ideas reshape civilisations while others — equally brilliant — fade into obscurity?

This site applies five analytical frameworks to understand idea propagation:

| Section | Framework | Core Question |
|---------|-----------|---------------|
| **Network Theory** | Graph theory, epidemiology | How does network position amplify ideas? |
| **Power Laws** | Statistical mechanics | Why do 1% of ideas capture 50% of influence? |
| **Memetics** | Evolutionary biology | What makes an idea a successful replicator? |
| **Information Theory** | Shannon, Kolmogorov | Why do the best ideas compress reality? |
| **Evolutionary Selection** | Darwinian dynamics | How do environments select for ideas? |
| **Case Studies** | Historical analysis | What patterns recur across dominant ideas? |
| **Predictions** | Signal detection | Can we identify tomorrow's dominant ideas today? |
| **Interactive** | Canvas API | See the dynamics in motion |

## Interactive Visualisations

Four Canvas API animations let you explore the dynamics hands-on:

1. **Network Cascade** — Watch an idea spread through a scale-free network. Adjust hub influence and density.
2. **Power Law Emergence** — See how preferential attachment creates extreme inequality from a simple rule.
3. **Evolutionary Competition** — Multiple ideas compete; trigger a paradigm shift and watch dominance invert.
4. **Compression Power** — Visualise the trade-off between theory simplicity and data fit (Occam's Razor).
